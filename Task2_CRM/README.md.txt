# Client Lead Management System (Mini CRM)

## 📌 Project Description
This is a full-stack CRM application built to manage client leads.  
Users can add, update, and delete leads with status tracking.

---

## 🚀 Features
- Add new leads
- Update lead status (New / Contacted / Converted)
- Delete leads
- View all leads
- MongoDB database integration

---

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MongoDB Atlas

---

## ▶️ How to Run (Local Setup)

1. Go to backend folder  
2. Run:
node server.js

3. Open `index.html` in browser  

---

## 🌐 Deployment Status
Frontend: Deployed on Netlify  
Backend: Deployed on Render  

---

## ⚠️ Note
If running locally, make sure MongoDB connection is active and backend server is running before using the frontend.
